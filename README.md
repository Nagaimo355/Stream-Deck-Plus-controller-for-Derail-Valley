# Stream-Deck-Plus-controller-for-Derail-Valley
Stream deck plusを用いたDerail Valley用コントローラー。SL以外のディーゼル/バッテリー機関車で使用可能です。

# 必要なもの
- Stream Deck Plus
- Stream Deck ソフトウェア
- AutoHotkey（v2.0）

# How to Use
１．
「StreamDeck_SupportFile.zip」をダウンロード＆解凍し、フォルダ「StreamDeck_SupportFile」をデスクトップに配置します。

２．
フォルダ「Derail Valley」の「260317_StreamDeckPlus_DerailValley.streamDeckProfilesBackup」をダブルクリックし、バックアップファイルを適用します。

３．
AutoHotkey（v2.0）をダウンロード・インストールします。
これによってキーボード自動操作スクリプトが動作可能になります。

ダウンロード：https://www.autohotkey.com/

４．
ゲーム「Derail Valley」のキーバインド設定をAHKスクリプトと同じにします。もしくは、ご自身のキーバインド設定に合わせてAHKスクリプトを変更します。

たぶんコレで使えるはずです。（GitHub初心者です。申し訳ございません。）
